# EndMinecraftUltra
支持最新协议库的MC服务器压测工具,基于MCProtocolLib<br>
只可用于服务器压力测试,否则造成的后果由使用者自行承担!<br>
完整的最新构建版本可在Releases中下载,如有问题请发起Issues<br>

#### 支持方式: ####
1.单IP MOTD压测<br>
2.集群假人压测<br>
3.集群假人Tab压测<br>
4.可配置的刷屏/自动执行命令<br>
5.目前支持最新的MC1.16.4版本<br>
